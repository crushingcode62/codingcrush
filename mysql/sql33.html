UNION

UNION : The Union operator joins the outputs of two or more queries (select statement) into a single set of rows and columns having distinct records. Every select statement within union must have the same number of columns. The columns must also have similar data types. The columns in every select statement must also be in same order.


Syntax:

SELECT column_name FROM table1 UNION SELECT column_name FROM table2 [order by clause];
