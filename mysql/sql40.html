DATE FUNCTIONS

Oracle databse stores dates in an internal numeric format, representing the century, year, month, day, hours, minutes and seconds. The default date display fromat is DD-MON-RR.


(i) SYSDATE

SYSDATE is a pseudo-column that returns the system's current date and time of types DATE. SYSDATE returns the date and time of the local databse.

Syntax :

SELECT SYSDATE from dual ;


(ii)DATE_ADD(d,n)

This function adds days to or from a date. It returns a date as result.

Syntax :

SELECT SYSDATE, DATE_ADD(d,n) from dual ;


(iii) DATEDIFF(d1,d2)

This function returns the number of months between two dates, d1 and d2.

Syntax :

SELECT DATEDIFF(d1,d2) from dual ;


(iv) DATE_SUB(m,n)

This function substract days to or from a date. It returns a date as result.

Syntax :

SELECT DATE_SUB(m,n) from dual ;


(v) DATE_FORMAT(d, format)

This function is used to change the date format.

Syntax :

SELECT DATE_FORMAT(d,format) from dual ;
