DEFAULT

Default:The default constraints is used to insert a default value into a column. The Default value will be added to all new records, if no other value is specified. The most common value is NULL. This can be used with columns not defined with a NOT NULL.


Syntax:

create table table_name
( column_name1 datatype, column_name2 datatype, column_name3 datatype DEFAULT 'value'.....);
