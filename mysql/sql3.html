Delete Database

We can delete/remove database using DROP databse command in MYSQL. It will delete the database along with all the tables,indexes and constraints permanently.
If we need to delete the database, we can use the drop command in mysql command client line. In this using drop the databse will be deleted.
In the following example we see that we have a database that is record. To remove this database we can use the drop command.

Syntax :

drop database databasename ;
