NOT NULL

Not NULL: A NOT NULL is a column level Constraints. We can defined not NULL to multiple column or single. When not NULL is defined the we cannot give NULL value in that column.

Syntax :

Create table table_name
(column_name1 datatype Constraints Constraints_name NOT NULL, column_name2 datatype);

Example:Create a table emp_det with not allow null value.

If we can enter the NULL value in the table the error will be show.
