Delete command

The DELETE statement is used to delete rows from a table. To delete rows from a table, the table must be in your own schema or you must have the delete privilege on the table.
Specific rows are deleted if you specify the where clause in the delete statement. You cannot delete a row that contains a primary key which is used as a foreign key in another table.

Syntax :

delete from table_name [where clause];
