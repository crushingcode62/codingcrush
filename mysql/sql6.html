DESC Command

DESCRIBE means to show the information in detail. In MYSQL, so we will use the DESC/DESCRIBE command to show the structure of our table,such as column names, constraints on column names,etc. To see the structure of the existing table we can use the desc(describe) command. Using this command many type of the columns are show like type, Null, key, default etc.

Syntax :

desc tablename ;
