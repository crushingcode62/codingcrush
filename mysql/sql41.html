CHARACTER FUNCTIONS

Character or text Functions are used to manipulate text strings. They accept string or character data as input and can return both character and numeric values. Character Functions can be divided into the following:
1.Case-manipulation Functions
2.Character-manipulation Functions

Case-manipulation Functions

These Functions convert case for character strings. Following are the three Case-manipulation funtions:

(i)LOWER(string)

This function converts mixed case or uppercase character strings to lowercase.

Syntax :

SELECT LOWER(string) from dual ;


(ii)UPPER(string)

This function converts mixed case or lowercase character strings to uppercase.

Syntax :

SELECT UPPER(string) from dual ;


(iii)INITCAP(string)

This function converts the first letter of each word to uppercase and remaining letters to lowercase.

Syntax :

SELECT INITCAP(string) from dual ;

Character-manipulation Functions

These Functions manipulate character strings. Following are the three Case-manipulation funtions:

(i)CONCAT(string)

This function joins two string values together. This function is similar to || Operator.

Syntax :

SELECT CONCAT('string1', 'string2') from dual ;


(ii)LENGTH(string)

This function returns the length of a string as a numeric value.

Syntax :

SELECT LENGTH(string) from dual ;


(iii)ASCII(string)

This function returns the decimal representation of the first character of the string.

Syntax :

SELECT ASCII(string) from dual ;


(iv) REPLACE(string, serch_str, replace_str)

This function returns string with every occurence of search_str replaced with replace_str.

Syntax :

SELECT REPLACE(string, serch_str, replace_str) from dual ;
