Select command

The most commonly used SQL command is SELECT statement. The select statement is used to query or retrieve data from a table in the database. A query may retrieve information from specified columns or from all of the columns in the table.

Syntax :

select column_name from table_name [where clause] ;


For access all columns :

select * from table_name [where clause];
