Use Database

For using the existing databases in mysql we can use the use command in MySQL command line client. Using this command we can use the particular database. We can use SQL command USE to select a particular database.
All the database names, table names and table fields name are case sensitive. You must have to use proper names while giving any SQL command.

Syntax :

use databasename ;
