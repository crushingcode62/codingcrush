UNIQUE:

Unique:It is a table level and also a column level constraints. The UNIQUE constraints uniquelly indentifies each row in a database. It ensures that the values entered into a column are unique.

Syntax(Unique at column-level):

create table table_name
( column_name1 datatype UNIQUE, column_name2 datatype,.....);


Syntax(Unique at table-level):

create table table_name
(column_name1 datatype , column_name2 datatype,.....,UNIQUE (column_name1));

Example: Create table emp_det which have allow to insert unique values.

Using the unique clause we cannot enter the duplicate value in that column. If we can enter the error will be show.
