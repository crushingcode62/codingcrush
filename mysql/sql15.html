Order by clause

Whenever we want to sort the records based on the columns stored in the tables of the SQL database, then we consider using the ORDER BY clause in SQL.
The ORDER BY clause in SQL will help us to sort the records based on the specific column of a table. This means that all the values stored in the column on which we are applying ORDER BY clause will be sorted, and the corresponding column values will be displayed in the sequence in which we have obtained the values in the earlier step. Using the ORDER BY clause, we can sort the records in ascending or descending order as per our requirement. The records will be sorted in ascending order whenever the ASC keyword is used with ORDER by clause. DESC keyword will sort the records in descending order.
If no keyword is specified after the column based on which we have to sort the records, in that case, the sorting will be done by default in the ascending order.

Syntax :

SELECT Column_Name1,..Column_NameN FROM Table_Name [where condition]
ORDER BY Column_Name ASC|DESC;
