MINUS

MINUS :Minus operator returns the rows unique to first query.It returns the remaining rows when the rows restrieved by the second query are subtracted from the rows retrieved by the first query.
The MINUS operator returns the unique element from the first table/set, which is not found in the second table/set.
The number and order of columns in all the select statements must be the same. The data types of the corresponding columns in both select statements must be same or convertible.


Syntax:

SELECT column_name FROM table1 MINUS SELECT column_name FROM table2 [order by clause];
