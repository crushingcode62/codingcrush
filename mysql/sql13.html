Where Clause

The where clause is used when you want to retrieve the specific information from a table excluding other irrelavant data. SQL offers a feature called WHERE CLAUSE, which we can use to restrict data that is retrieved. Where clause can be used along with select, delete and update statement.


Syntax :

select column_name from table_name where condition;

Output:



In where clause we can use multiple operators in the where clause condition.
Types of operators in SQL:

  • Comparison Operator
  • Logical Operator
  • Special Operator
