NUMBER FUNCTIONS

Number or numeric functions are used to perform operations on numbers. They accept numeric values as input and return numeric values as output.Dual table is a single row and single column dummy table provided by Oracle. Few of the numeric functions are :


(i) ABS(n)

This function returns absolute value of the value passed.

Syntax :

SELECT ABS(n) from dual ;


(ii)CEIL(n)

This function finds smallest integer greater than or equal to n. Where n can be either column name or expression.

Syntax :

SELECT CEIL(n) from dual ;


(iii) FLOOR(n)

This function find the largest integer less than or equal to the value n.

Syntax :

SELECT FLOOR(n) from dual ;


(iv) MOD(m,n)

This function returns the remainder of m divided by n.

Syntax :

SELECT MOD(m,n) from dual ;


(v) POWER(m,n)

This function returns m raised to the power n. The second argument n must be an integer.

Syntax :

SELECT POWER(m,n) from dual ;


(vi) ROUND(m,[n])

This function rounds the value m to n decimal places. If n is omitted, then it is rounded to no decimal places.

Syntax :

SELECT ROUND(m,n) from dual ;
