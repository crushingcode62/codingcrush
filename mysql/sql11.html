Update command

The UPDATE statement is used to update or modify the existing rows in a table. The update statement modifies specific rows if the where clause is specified.
If you omit the where clause, the all the rows in the table are modified. Values of a single column or a group of columns can be updated. The update statement sets specific values for each column required to change.

Syntax :

update table_name
set column_name= value [where condition];
