UNION ALL

UNION ALL: The Union all operator joins the outputs of two or more queries (select statement) into a single set of rows and columns without the removal of any duplicates.
Like, MYSQL union operator it is also a useful command in MYSQL database to combine more than two of the output set provided by the use of select queries. But there is slightly a difference in their work which makes them two different commands used in MYSQL for theirr respective purposes. The union all operator allows the records with duplicate values.


Syntax:

SELECT column_name FROM table1 UNION ALL SELECT column_name FROM table2 [order by clause];
