Insert Command

The INSERT statement is used to insert new rows or records into a table. Values can be inserted for all the columns or for the selected columns of the table.

1.Insert into single row
Syntax:
insert into table_name (column_name) values (value);

2.Insert into multiple rows
Syntax:
insert into table_name values(value1, value2, value3,...);
