Create Command

The CREATE is a data definition language command. The create command is used to create tables to store data. Tables are owned by the user who creates them. The column names in the table must be unique.
For creating the table in a particular database, first of all we can change the current database using use command and after that we can create a table using create table command.

Syntax :

create table tablename(colname1 datatype, colname2 datatype, ......) ;
