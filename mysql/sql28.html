JOINS

SQL joins are used to relate information in different tables. Joins are used to combine columns from different tables. The connection between tables is established through the WHERE clause, called SQL joins condition. With joins, the informationfrom any number of tables can be accessed.

Syntax :

Select table1.column, table2.column,........ from table1, table2..... where table1.column1= table2.column2;

Types of Joins:

  1. Equi Joins
  2. Cartesian Joins
  3. Left Outer Joins
  4. Right Outer Joins
  5. Self Joins
