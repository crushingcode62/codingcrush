USER DEFINED EXCEPTIONS

PL/SQL allows you to define your own exceptions according to the need of your program. A user-defined exception must be declared and then raised explicitly, using either a RAISE statement or the procedure DBMS_STANDARD.RAISE_APPLICATION_ERROR.

Steps to be followed to use User defined Exceptions:

  • They should be explicity declard in the declaration section. (Declaring an Exception)
  • They should be explicitly raised in the Execution section. (Raising an Execution)
  • They should be handled by referencing the user-defined exception name in the exception section. (Handling an Exception)

Syntax:

DECLARE
user_exception EXCEPTION;
BEGIN
statement(s);
IF condition THEN
RAISE user_exception;
END IF;
EXCEPTION
WHEN user_exception THEN
User defined statement (action) will be taken;
END;
