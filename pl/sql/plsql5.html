Conditional Control

The structure of the pl/sql if statement is similar to the structure of if statements in other languages. It allows pl/sql to perform actions selectively based on conditions. There are three forms of if statements:

  1. If then statement
  2. If-else then statement
  3. Elseif then statement


1.If Then Statement

The if then statement either runs or skips a sequence of one or more statements, depending on a condition.

Syntax:

IF condition THEN
statements
END IF;


Example: Find the biggest number between two numbers using IF THEN statement.


2.If Then Else Statement

In if then else statement, if the value of condition is true, the statements run; otherwise, the else statement run.

Syntax:

IF condition THEN
statements
ELSE
else_statements
END IF;


Example: Write a procedure to check the x is eligible for vote or not using IF THEN ELSE statement.


3. If Then Elsif Statement

The if then elsif statement runs the first statements for which condition is true. Remaining conditions are not evaluated. If no condition is true , the else_statements run, if they exist; otherwise, the if then elsif statement does nothing.

Syntax:

IF condition1 THEN
statements1
ELSIF condition2 THEN
statement2
ELSIF condition3 THEN
statement3
[ELSE
else_statements]
END IF;


Example:Find the biggest number between three numbers using IF THEN ELSIF statement.
