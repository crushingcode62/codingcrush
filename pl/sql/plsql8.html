SEQUENTIAL CONTROL

By default, all PL/SQL blocks are executed in a top-down sequantial process. The process begins with a BEGIN statement and terminates with an END statement. So, to change the sequance of execution of statements, we can use GOTO unconditional statement.


GOTO STATEMENT

The pl/sql GOTO statement is a Sequential control structure available in oracle. The GOTO statement immediately transfers program control unconditionally to a named statement label or block label. The statement or label name must be unique in the block.

Syntax:

GOTO << label_name >>;
statement1;
<< label_name>>
statement2;
