UNNAMED EXCEPTIONS

Oracle doesn’t provide name for some system exceptions called unnamed system exceptions.These exceptions don’t occur frequently.These exceptions have two parts code and an associated message. When we want to handle the unnamed exception then first we will declare exception type variable with the exception.

Syntax:

Declare
exp_var exception;
(Now, link this variable with the exception SQL code by using the following :)
pragma exception_init (exp_var, -code);
BEGIN
executable_statement;
EXCEPTION
WHEN exp_var THEN
exception-handling-statements;
END;
