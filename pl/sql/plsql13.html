CURSOR WITH PARAMETERS

Parameters allow values to be passed to a cursor when it is opend, and used within the query when it executes. Using parameters explicit cursor may be opened more than once in a block, returning a different active set each time.
Each formal parameter in the cursor declaration must have a corresponding actual parameter in the open statement. Parameter datatypes are the same as those for scalar variables, but you do not give them sizes. The parameters are treated like pl/sql variables within the cursor's query expression.

Syntax:

CURSOR cursor_name
[parameter_name datatype]
IS
select statement;
While opening the cursor, parameters values are passed.
OPEN cursor_name(parameter_value....);
